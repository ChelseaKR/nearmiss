# Pre-registration artifacts (EXP-16)

This directory holds the output of `nearmiss preregister` and `nearmiss
score-preregistration` — see [`docs/PREREGISTRATION.md`](../../../docs/PREREGISTRATION.md).

It is intentionally empty as shipped: no registration has been committed here
yet, because no real one has been made. A committed registration artifact
in this directory is a claim that a real, unmodified evaluation window is (or
was) open against real accumulating data, under a scoring rule that a
statistician has signed off on (see
`docs/preregistration/scoring-rule-signoff.json`). Do not add a file here that
doesn't meet that bar — see the "SME gate" section of
`docs/PREREGISTRATION.md` for what "meets that bar" means.
