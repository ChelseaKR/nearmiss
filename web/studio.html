01 · Source readiness

Check the evidence before running an analysis.

This quick audit recognizes common report fields, checks whether the observation window and location are usable, and estimates whether a privacy floor would suppress grouped results. It is a readiness screen, not a statistical result.

CSV and JSON are supported. Useful fields include latitude/longitude or address, date or timestamp, hazard type, and a segment or corridor identifier.
