# Decision Dossier template

Use this template to turn a street-safety concern into a reviewable decision
artifact. It is deliberately usable before the local-first Studio is built. Copy
it into a dated working document, complete only the sections supported by your
evidence, and retain the empty sections when evidence is missing.

This is a planning and communication template, not a substitute for local
engineering judgment, legal review, a traffic study, or a causal evaluation.

State one action that a named person or body can take. “Make this street safer”
is a goal, not a decision request. “Fund a field audit before the 2027 capital
plan” is a decision request.

## 2. Claim boundary

Choose the strongest statement the evidence supports. Delete stronger statements.

- **Tier 0 — coverage gap:** Evidence is insufficient to rank this corridor.
- **Tier 1 — community signal:** Repeated community reports warrant investigation.
- **Tier 2 — elevated rate:** The observed report rate is elevated in the stated
  analysis window.
- **Tier 3 — triangulated priority:** Independent evidence supports prioritizing
  this corridor for the stated next action.

**Permitted claim:**

**This dossier does not establish:** danger, fault, causation, the likely effect
of a treatment, or that a treatment worked. Add any local limitations here.
